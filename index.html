Collapse Link
Some placeholder content for the collapse component. This panel is hidden by default but revealed when the user activates the relevant trigger.

This example is based off of code provided in the following blog by Jason Butz. It has been updated to be object-oriented. Collapse.getInstances() and Collapse.getInstance() were added as static methods. show(), hide(), and destroy() were added as instance methods. destroy() removes the associated event listeners and the the corresponding instance reference stored on the prototype.

Collapse instances were created for the nav menu as well as the example in the overview.
